# .claude

Project-scoped Claude Code configuration.

- `settings.json` — shared, checked-in settings (permissions, env, hooks).
- `settings.local.json` — personal overrides; git-ignored.
- `commands/` — project slash commands (e.g. `/check`, `/bootstrap`).
- `agents/` — project sub-agents (optional, add as needed).

See https://docs.claude.com/en/docs/claude-code/settings for the full schema.
